🦐 N+1 LAB 采购助手 - 插件测试

测试步骤：

  1. 打开淘宝商品页面（任意商品）
  2. 在 URL 后面添加 ?n1_auto_add=1&n1_qty=3
  3. 刷新页面
  4. 检查是否出现"🦐 小虾虾 v6.8"悬浮窗

✅ 预期结果：

  • 页面右上角出现"🦐 小虾虾 v6.8 [智能重试版]"悬浮窗
  • 显示"🚀 正在初始化..."
  • 自动填写数量输入框为 3
  • 显示"✅ 数量已设为 3，请点击【加入购物车】按钮"

⚠️ 如果没出现悬浮窗：

  1. 打开 Chrome 开发者工具（F12）
  2. 切换到 Console 标签
  3. 刷新页面
  4. 检查是否有"🦐 N+1 自动化 v6.8 启动"日志
  5. 如果没有日志，说明插件未加载，请刷新扩展

🔧 刷新扩展步骤：

  1. 打开 chrome://extensions/
  2. 找到"N+1 LAB 采购助手"
  3. 点击刷新按钮 🔄
  4. 重新打开淘宝测试页面
