**添加时间**: 2026-04-04 16:31  
**版本**: v1.0

## ✅ 功能说明

### 退出登录按钮

**位置**: 左侧导航栏底部  
**图标**: 🚪  
**颜色**: 红色（#ff6b6b）

**功能**:
- ✅ 点击后弹出确认对话框
- ✅ 确认后清除 API Key
- ✅ 自动刷新页面
- ✅ 需要重新输入 API Key 才能访问

---

## 🎯 使用流程

### 退出登录
1. 点击左侧导航栏底部的"🚪 退出登录"
2. 弹出确认对话框：
   ```
   确定要退出登录吗？
   
   退出后需要重新输入 API Key 才能访问系统。
   ```
3. 点击"确定"
4. 系统清除 API Key 并刷新页面
5. 显示 API Key 输入界面

### 重新登录
1. 刷新页面后自动显示登录界面
2. 输入 API Key（`n1lab2026`）
3. 点击"登录"
4. 进入系统

## 🔧 技术实现

### HTML 结构
```html
<div style="padding:16px; border-top:1px solid #ffffff20;">
    <div class="nav-item" onclick="logout()" style="color:#ff6b6b;">
        <span>🚪</span>
        <span class="nav-text">退出登录</span>
    </div>
</div>
```

### JavaScript 函数
```javascript
function logout() {
    if (confirm('确定要退出登录吗？\n\n退出后需要重新输入 API Key 才能访问系统。')) {
        // 清除 API Key
        localStorage.removeItem('n1_api_key');
        window.apiKey = '';
        
        // 刷新页面
        alert('✅ 已退出登录\n\n请刷新页面重新登录。');
        location.reload();
    }
}
```

## 📝 注意事项

1. **退出后数据不会丢失**
   - 只是清除认证信息
   - 数据库数据不受影响

2. **多用户场景**
   - 退出后可以切换其他用户
   - 每个用户有独立的 API Key

3. **安全性**
   - 建议定期退出登录
   - 公共电脑使用后务必退出
