**修复时间**: 2026-04-04  
**负责人**: 小虾虾 🦐  
**问题报告**: Shuni 反馈产品中心页面零件包售价为 0，改价按钮无反应

---

## 🐛 问题清单

### 1. 改价按钮点击无反应
**原因**: 前端代码调用了 `editPrice()` 和 `savePrice()` 函数，但这些函数未定义

### 2. 无法编辑库存
**原因**: 
- 数据库缺少 `kit_stock_quantity`（零件包库存）字段
- 前端没有库存编辑 UI
- 后端没有库存更新 API

### 3. 库存管理不完善
**需求**: 需要区分**零件包库存**和**产品库存**（整体成品库存）

---

## ✅ 修复内容

### 1. 数据库迁移

**文件**: `backend/add_kit_stock_field.py`

```python
# 添加零件包库存字段
ALTER TABLE products 
ADD COLUMN kit_stock_quantity INTEGER DEFAULT 0

# 同步现有库存数据
UPDATE products 
SET kit_stock_quantity = stock_quantity 
WHERE stock_quantity IS NOT NULL
```

**执行结果**: ✅ 成功添加字段并同步数据

---

### 2. 后端 API 更新

#### 2.1 更新 Product 模型

**文件**: `backend/main.py` (第 36-47 行)

```python
class Product(Base):
    __tablename__ = "products"
    # ... 其他字段
    stock_quantity = Column(Integer, default=0)  # 产品库存（整体）
    kit_stock_quantity = Column(Integer, default=0)  # 零件包库存（新增）
    # ...
```

#### 2.2 更新 BOM 成本 API

**接口**: `GET /admin/products/bom-costs`

**新增返回字段**:
- `stock_quantity`: 产品库存（整体成品）
- `kit_stock_quantity`: 零件包库存

#### 2.3 新增库存更新 API

**接口**: `PUT /admin/products/{product_id}/stock`

**参数**:
- `kit_stock_quantity`: 零件包库存（可选）
- `stock_quantity`: 产品库存（可选）

**示例**:
```bash
curl -X PUT "http://43.156.225.39:8080/admin/products/1/stock?kit_stock_quantity=50&stock_quantity=100" \
  -H "X-API-Key: n1lab2026"
```

## 🔧 使用方法

### 1. 修改产品价格

1. 进入产品中心页面
2. 找到目标产品
3. 点击 **✏️ 改价** 按钮
4. 输入新的零件包售价和零售价
5. 点击 **💾 保存**

### 2. 修改产品库存

1. 进入产品中心页面
2. 找到目标产品
3. 点击 **📦 库存** 按钮
4. 分别输入零件包库存和产品库存
5. 点击 **💾 保存**

### 3. 批量初始化库存

```bash
python3 add_kit_stock_field.py
```
