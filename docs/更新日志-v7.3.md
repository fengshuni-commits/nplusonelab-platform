## 🔧 安装方法

1. **解压插件**
   ```bash
   tar -xzf n1_helper_v7.3.tar.gz
   ```

2. **Chrome 扩展管理**
   - 打开：`chrome://extensions/`
   - 开启"开发者模式"

3. **加载插件**
   - 点击"加载已解压的扩展程序"
   - 选择 `extension` 文件夹
   - 确认版本显示为 **v7.3**
