#### 3. 价格计算函数
```javascript
function calculateTotal() {
    const quantity = parseInt(document.getElementById('order-quantity').value) || 1;
    const price = parseFloat(document.getElementById('order-price').value) || 0;
    const discount = parseInt(document.getElementById('order-discount').value) || 100;
    
    const originalTotal = price * quantity;
    const discountAmount = originalTotal * (1 - discount / 100);
    const finalTotal = originalTotal - discountAmount;
    
    document.getElementById('order-original-total').innerText = '¥' + originalTotal.toFixed(2);
    document.getElementById('order-discount-amount').innerText = '-¥' + discountAmount.toFixed(2);
    document.getElementById('order-total').innerText = '¥' + finalTotal.toFixed(2);
    
    // 根据折扣显示不同颜色
    const totalElement = document.getElementById('order-total');
    if (discount < 100) {
        totalElement.style.color = '#fa8c16'; // 橙色
    } else {
        totalElement.style.color = '#1890ff'; // 蓝色
    }
}
```

#### 4. 创建订单时处理折扣
```javascript
async function confirmCreateOrder() {
    // ... 其他代码 ...
    
    const discount = parseInt(document.getElementById('order-discount').value) || 100;
    const originalPrice = parseFloat(selectedOption.dataset.price) || 0;
    const discountedPrice = originalPrice * (discount / 100);
    const totalAmount = discountedPrice * quantity;
    
    // 在备注中添加折扣信息
    let finalRemark = remark;
    if (discount < 100) {
        const discountInfo = ` [折扣：${discount}%, 优惠：¥${(originalPrice * quantity * (1 - discount/100)).toFixed(2)}]`;
        finalRemark = remark ? remark + discountInfo : discountInfo;
    }
    
    // 发送到后端
    await fetch(`${API_BASE}/orders/`, {
        method: 'POST',
        body: JSON.stringify({
            // ...
            unit_price: discountedPrice, // 使用折扣后的单价
            total_amount: totalAmount, // 使用折扣后的总额
            remark: finalRemark // 包含折扣信息
        })
    });
}
```

## 📝 注意事项

### 折扣范围
- **最小值**: 0% (免费赠送)
- **最大值**: 100% (原价)
- **默认值**: 100% (无折扣)

### 价格精度
- **单价**: 保留 2 位小数
- **总额**: 保留 2 位小数
- **优惠金额**: 保留 2 位小数

### 数据记录
- **单价**: 存储折扣后的实际单价
- **总额**: 存储折扣后的实际总额
- **备注**: 自动添加折扣信息（仅在有折扣时）

### 视觉反馈
