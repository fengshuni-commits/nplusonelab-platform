## ✅ 功能说明

### 登录验证机制

**认证方式**: API Key  
**存储方式**: localStorage  
**默认 API Key**: `n1lab2026`

**工作流程**:
1. 用户首次访问 → 弹出 API Key 输入框
2. 输入正确的 API Key → 保存到 localStorage
3. 后续访问 → 自动从 localStorage 读取
4. 点击退出登录 → 清除 localStorage，重新输入

## 🔧 技术实现

### 核心代码

**1. 读取 API Key**
```javascript
let key = localStorage.getItem('n1_api_key') || '';
```

**2. 检查登录状态**
```javascript
function checkAuth() {
    if (!key) {
        showLoginPrompt();
        return false;
    }
    return true;
}
```

**3. 显示登录提示**
```javascript
function showLoginPrompt() {
    const apiKey = prompt('请输入 API Key 以访问系统：', '');
    if (apiKey) {
        key = apiKey;
        localStorage.setItem('n1_api_key', apiKey);
        location.reload();
    }
}
```

**4. 退出登录**
```javascript
function logout() {
    localStorage.removeItem('n1_api_key');
    key = '';
    showLoginPrompt();
}
```
