## 🔧 技术实现

### API 调用修复

**修改前**:
```javascript
fetch(`${API_BASE}/admin/products/bom-costs`, {
    headers: { 'X-API-Key': key }  // ❌ key 未定义
})
```

**修改后**:
```javascript
fetch(`${API_BASE}/admin/products/bom-costs`, {
    headers: { 'X-API-Key': 'n1lab2026' }  // ✅ 固定 API Key
})
```

### 权限控制

**代码**:
```javascript
window.addEventListener('DOMContentLoaded', () => {
    if (currentUser) {
        // 根据角色显示/隐藏菜单
        if (currentUser.role !== 'admin') {
            // 普通用户隐藏用户管理菜单
            const userMenuItem = document.querySelector(
                '.nav-item[onclick="switchTab(\'users\', this)"]'
            );
            if (userMenuItem) {
                userMenuItem.style.display = 'none';
            }
        }
    }
});
```
