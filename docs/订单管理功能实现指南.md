# 订单管理功能实现指南

**创建时间**: 2026-04-04 21:09  
**版本**: v1.0

---

## ✅ 已完成功能

### 1. **数据库表** ✅
- ✅ `orders` 表 - 订单基本信息
- ✅ `order_items` 表 - 订单明细（关联 BOM）

### 2. **后端 API** ✅
- ✅ `GET /orders/` - 获取订单列表
- ✅ `POST /orders/` - 创建订单
- ✅ `GET /orders/{order_id}` - 获取订单详情
- ✅ `PUT /orders/{order_id}/status` - 更新订单状态
- ✅ `DELETE /orders/{order_id}` - 删除订单

### 3. **订单类型** ✅
- ✅ **零件包订单** (`kit`) - 关联零件包 BOM
- ✅ **整体产品订单** (`complete`) - 关联整体 BOM

## 📊 数据库结构

### orders 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_no | TEXT | 订单号（唯一） |
| customer_name | TEXT | 客户名称 |
| customer_contact | TEXT | 客户联系方式 |
| order_type | TEXT | 订单类型（kit/complete） |
| product_id | INTEGER | 产品 ID |
| product_sku | TEXT | 产品 SKU |
| product_name | TEXT | 产品名称 |
| quantity | INTEGER | 数量 |
| unit_price | REAL | 单价 |
| total_amount | REAL | 总额 |
| status | TEXT | 状态（pending/processing/completed/cancelled） |
| remark | TEXT | 备注 |
| created_by | INTEGER | 创建人 ID |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### order_items 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_id | INTEGER | 订单 ID（外键） |
| bom_item_id | INTEGER | BOM 零件 ID（外键） |
| part_name | TEXT | 零件名称 |
| specification | TEXT | 规格 |
| quantity | REAL | 数量 |
| unit_price | REAL | 单价 |
| total_price | REAL | 总价 |
| taobao_link | TEXT | 淘宝链接 |
| status | TEXT | 采购状态 |

## 🎯 创建订单流程

### 1. 填写订单信息
```
客户名称：[输入框]
联系方式：[输入框]
订单类型：○ 零件包  ○ 整体产品
选择产品：[下拉选择]
数量：[输入框]
备注：[文本框]
```

### 2. 系统自动处理
```
1. 生成订单号（ORD202604040001）
2. 计算总价（单价 × 数量）
3. 如果是零件包订单：
   - 查询产品 BOM
   - 创建订单明细（关联 BOM 零件）
4. 保存订单到数据库
```

### 3. 订单详情
```
订单号：ORD202604040001
客户：张三
类型：零件包
产品：AP2401-SKU1
数量：5
总额：¥1000.00
状态：待处理

订单明细（BOM 零件）:
- 4040 型材 × 10
- 一字连接件 × 15
- ...
```
