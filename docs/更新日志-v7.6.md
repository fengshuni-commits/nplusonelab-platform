#### 新增功能
- ✅ 订单列表添加"🛒 零部件下单"按钮

#### 2. orderParts() 函数
```javascript
async function orderParts(orderId, productId, orderType) {
    // 1. 获取订单详情和 BOM 清单
    const orderData = await fetch(`${API_BASE}/orders/${orderId}`, {
        headers: { 'X-API-Key': 'n1lab2026' }
    }).then(r => r.json());
    
    const bomItems = orderData.items || [];
    
    // 2. 构建采购清单
    const partsList = bomItems.map(item => ({
        link: item.taobao_link,
        quantity: Math.ceil(item.quantity)
    }));
    
    // 3. 显示确认弹窗
    if (!confirm('确认采购？')) return;
    
    // 4. 打开淘宝页面，附带采购参数
    const purchaseUrl = `${firstPart.link}?n1_auto_add=1&n1_order_id=${orderId}&n1_parts=${encodeURIComponent(JSON.stringify(partsList))}`;
    window.open(purchaseUrl, '_blank');
    
    // 5. 更新订单状态
    await fetch(`${API_BASE}/orders/${orderId}/status`, {
        method: 'PUT',
        body: JSON.stringify({ status: 'processing' })
    });
}
```

## 📋 采购插件集成

### 采购插件 v7.4+ 功能

#### URL 参数识别
采购插件会自动识别以下 URL 参数：
- `n1_auto_add=1` - 启用自动加购模式
- `n1_order_id=1` - 关联订单 ID
- `n1_parts=[...]` - 零件清单 JSON

#### 自动加购流程
1. **页面加载**: 检测 URL 参数 `n1_auto_add=1`
2. **解析清单**: 读取 `n1_parts` 参数中的零件列表
3. **显示清单**: 在插件界面显示采购清单
4. **等待确认**: 用户点击"开启全自动采购"
5. **批量加购**: 依次打开每个零件页面并加购
6. **完成提示**: 显示加购结果统计

## 📝 注意事项

### 前置条件
1. ✅ 已安装 N+1 采购插件 v7.4+
2. ✅ 已登录淘宝账号
3. ✅ 浏览器允许弹出窗口
4. ✅ 购物车有足够空间（淘宝购物车上限 120 件）

### 兼容性
- **采购插件版本**: v7.4 或更高
- **浏览器**: Chrome / Edge / Firefox
- **淘宝域名**: taobao.com / tmall.com

### 错误处理
- **无 BOM 数据**: 提示"该订单没有 BOM 零件清单"
- **无效链接**: 跳过无淘宝链接的零件
- **弹窗拦截**: 提示"浏览器拦截了弹窗"
- **加购失败**: 手动复制链接采购
