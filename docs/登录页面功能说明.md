## 🔧 技术实现

### 登录验证流程

**login.html**:
```javascript
// 检查是否已登录
const savedKey = localStorage.getItem('n1_api_key');
if (savedKey) {
    window.location.href = '/';  // 已登录，跳主页
}

// 处理登录
function handleLogin(event) {
    const apiKey = document.getElementById('api-key').value;
    localStorage.setItem('n1_api_key', apiKey);
    window.location.href = '/';  // 跳转主页
}
```

**index.html**:
```javascript
// 页面加载时检查
window.addEventListener('DOMContentLoaded', () => {
    if (!key) {
        window.location.href = '/login.html';  // 未登录，跳登录页
    }
});
```

**logout()**:
```javascript
function logout() {
    localStorage.removeItem('n1_api_key');
    window.location.href = '/login.html';  // 清除并跳转
}
```
