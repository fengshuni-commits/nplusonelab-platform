## 🚀 安装步骤

### 方式一：加载已解压的扩展（推荐）

1. **解压扩展文件**
   ```bash
   # 复制到新目录
   mkdir -p /tmp/n1-extension-v6.7
   cp /root/.openclaw/workspace/projects/n1-lab-automation/frontend/n1_helper_v6.7_fixed.zip /tmp/
   cd /tmp && unzip n1_helper_v6.7_fixed.zip -d n1-extension-v6.7
   ```

2. **打开 Chrome 扩展管理**
   - 地址栏输入：`chrome://extensions/`
   - 或：菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 右上角切换"开发者模式"为开启状态

4. **加载扩展**
   - 点击"加载已解压的扩展"
   - 选择 `/tmp/n1-extension-v6.7` 文件夹
   - 扩展图标应出现在浏览器工具栏

### 方式二：直接拖入 CRX（需要打包）

如果 Chrome 支持直接安装未打包的扩展，可以直接拖入 zip 文件。

## 🐛 故障排查

### 问题 1: 扩展未激活
**现象**: 打开管理后台后没有看到 "N+1 采购助手 v6.7 已激活" 日志

**排查**:
1. 检查扩展是否启用：`chrome://extensions/`
2. 检查扩展权限是否授予
3. 检查网址是否匹配：`http://43.156.225.39:8080/` 或 `http://localhost:8080/`
4. 刷新页面重试

### 问题 2: BOM 按钮不出现
**现象**: 点击"拆解 BOM"后弹窗正常，但没有红色采购按钮

**排查**:
1. 检查 BOM 弹窗 ID 是否为 `bom-modal`
2. 检查是否有淘宝链接（`#bom-body a`）
3. 刷新页面重新打开 BOM 弹窗
4. 检查浏览器控制台是否有 JS 错误

### 问题 3: 无法自动加购
**现象**: 打开淘宝页面后，悬浮窗一直显示"扫描按钮中..."

**排查**:
1. 检查商品是否有货
2. 检查是否需要先选择规格（颜色/尺寸）
3. 检查页面是否完全加载
4. 手动点击"加入购物车"测试是否正常
5. 查看控制台日志定位问题

### 问题 4: 队列中断
**现象**: 加购一个商品后，没有自动打开下一个

**排查**:
1. 检查 background.js 是否正常工作
2. 查看扩展管理页面的错误日志
3. 检查是否所有标签页都被正确关闭
4. 重启扩展后重试
